# Validierungs-Reports (tägliche Automatisierung)

**Zweck**: Tägliche Reports der automatisierten Compliance-Prüfung (Gesetzeslage, Datum+Quelle, Links, Dokumentenstandard). Jeder Lauf erzeugt eine Datei `YYYY-MM-DD.md` mit Stufen Kritisch / Warnung / Info.

**Referenzen**: [Prozess Compliance-Automation & Nachvollziehbarkeit](../prozess_compliance_automation_und_nachvollziehbarkeit.md), [Gesetzeslage](../gesetzeslage_stand_280126.md).

**Stand**: 28.01.2026

---

## Format Report (YYYY-MM-DD.md)

- **Datum** des Laufs
- **Stufe** (Kritisch / Warnung / Info) pro Befund
- **Kurzbeschreibung** + **Quelle** + **Empfehlung**
- Optional: Link zu Issue (falls erstellt)

---

*Reports werden durch den täglichen Automatisierungs-Lauf erzeugt. Keine manuellen Platzhalter nötig.*
